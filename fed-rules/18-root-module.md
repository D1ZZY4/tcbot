**Usage Rules**:
- **NO Misleading Sales**: Selling root modules with labels like "VVIP," "PREMIUM," "EXCLUSIVE," or similar terms is strictly prohibited.
- **NO Harmful Modules**: Creating or distributing root modules that can cause device damage, brick, bootloop, or system instability is strictly prohibited. Developers are responsible for testing their modules before release.
- **NO Claiming Others' Work**: Do not claim ownership of root modules you did not create. Always credit the original creator(s).
- **Commercial Use**: Only permitted if you are the original creator OR the module's license explicitly allows commercial use and you comply with its terms.
- **License Compliance**: This rule is connected to licensing rules. If a module contains open-source code, you must comply with the original license terms (attribution, copyright notice, etc.).

**Violations & Sanctions**:
- Misleading sales ("VVIP" etc.) = **IMMEDIATE FBAN**.
- Harmful modules (brick/damage) = **IMMEDIATE FBAN + Blacklist**.
- Other violations = Warnings, sanctions, or FBAN depending on severity.

**Report to**: @TranssionCore1 (with valid proof).
